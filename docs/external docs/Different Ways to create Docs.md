## Using the Vs Code Extension Editor

This a full rich text editor built in using to Dev-Docs.  It is very similar to what you could expect from a notion type interface.  This editor has benefits built in like AI helpers, out of the box styling options, and out the box compoent selections so you do not have to remember shorthand.

## Using Dev-Docs Markdown

Dev-Docs has it's own custom markdown that gives you more flexability than the out-of-box markdown in docusaurus that allows you to style your markdown and use custom components without needing to use React.

## Using Docusaurus Markdown

At's it core Dev-Docs external documenation framework is built around all the great functionality of Docusaurus.  If you want to use out the box docusaurus features and markdown syntax you can do that.
